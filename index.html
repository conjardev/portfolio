Living life in code

I have been writing code as long as I can remember, whether it's making websites on the old family computer with my dad or making Scratch games when I was younger.

Now I am a full stack website developer building powerful web-based applications on a LAMP stack (Linux, Apache, MySQL, PHP), I am however always looking for new ways to enhance my knowledge and try new things.

I build modern and intuitive web frontends, with powerful microinteractions designed to make mundane tasks more interesting.

I use Adobe Illustrator to design vector art and graphics for my projects, and I use Adobe XD to plan the design and user flow of my web pages.

I have also started experimenting with Figma which has proven to be a useful tool and has helped to design this site.
